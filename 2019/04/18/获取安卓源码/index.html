获取安卓源码

发表于 04-18-2019 | 更新于 04-20-2019 | 分类于 Android ， Python

获取安卓源码的途径大致分为三个：

  1. repo；
  2. 网盘流；
  3. 使用自动化脚本获取； 每种途径的特点：
  4. repo：谷歌官方推荐使用，功能完善而且强大，基于 Python 2.7，但是克隆的源码十分巨大，高版本的系统源码高达上百G；
  5. 网盘流：简单粗暴，不过经常遇到网盘限速问题；
  6. 自动化脚本：灵活自由，需要写脚本；

网络上能找到的许多克隆源码的 Python 脚本一般都十分简陋，只能克隆 master 分支，并且无法指定深度等等

  1. 环境： python3.7.2，git（正确配置环境变量）；
